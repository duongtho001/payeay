🔑 API Key

API Key của bạn

Dùng API key này để tích hợp PayEay vào website hoặc backend của bạn.

Chưa có API Key — nhấn "Tạo mới" để sinh key
⚠️ Giữ API key bí mật. Nếu bị lộ, hãy nhấn "Tạo mới" để thay thế.

🌐 Base URL

https://api.payeay.vn/v1

📡 Endpoints

GET
/api/transaction/check

Kiểm tra giao dịch mới nhất khớp keyword

Tham sốKiểuMô tả
api_keystringAPI key của bạn *
keywordstringKeyword cần kiểm tra (VD: "ad", "ORDER123")
amountnumberSố tiền cần khớp (tùy chọn)
✅ Response thành công:
{
  "success": true,
  "data": {
    "message": "VCB: TK 0123xxx nhận +500,000 VND. ND: ad THANHTOAN123",
    "type": "notification",
    "source": "Vietcombank",
    "amount": 500000,
    "keyword": "ad",
    "timestamp": "2026-04-13T12:30:00Z"
  }
}
POST
/api/webhook/receive

App Android gửi thông báo về server (tự động)

Tham sốKiểuMô tả
api_keystringAPI key *
messagestringNội dung tin nhắn/thông báo *
typestring"sms" hoặc "notification"
sourcestringNguồn (VD: "Vietcombank")
GET
/api/transactions/list

Lấy danh sách giao dịch gần đây

Tham sốKiểuMô tả
api_keystringAPI key *
limitnumberSố giao dịch (mặc định: 20)
keywordstringLọc theo keyword (tùy chọn)

💻 Ví dụ tích hợp

curl -X GET "https://api.payeay.vn/v1/api/transaction/check?api_key=YOUR_API_KEY&keyword=ad&amount=500000"
// Kiểm tra giao dịch mới nhất
const response = await fetch(
  'https://api.payeay.vn/v1/api/transaction/check?' +
  new URLSearchParams({
    api_key: 'YOUR_API_KEY',
    keyword: 'ORDER123',
    amount: 500000
  })
);
const data = await response.json();

if (data.success) {
  console.log('Giao dịch khớp:', data.data.message);
  // Kích hoạt đơn hàng
} else {
  console.log('Chưa có giao dịch khớp');
}
<?php
// Kiểm tra giao dịch
$apiKey = 'YOUR_API_KEY';
$keyword = 'ORDER123';
$amount = 500000;

$url = "https://api.payeay.vn/v1/api/transaction/check?" . http_build_query([
    'api_key' => $apiKey,
    'keyword' => $keyword,
    'amount'  => $amount
]);

$response = json_decode(file_get_contents($url), true);

if ($response['success']) {
    echo "Giao dịch: " . $response['data']['message'];
    // Kích hoạt đơn hàng
} else {
    echo "Chưa có giao dịch khớp";
}
?>

🔔 Webhook Callback

Khi có giao dịch khớp keyword, PayEay sẽ gửi request đến webhook URL bạn đã cấu hình:

📤 Webhook payload gửi đến URL của bạn:
GET https://your-domain.com/webhook?message=VCB:+TK+0123xxx...&type=notification&source=Vietcombank&api_key=YOUR_KEY&keyword=ad&amount=500000×tamp=2026-04-13T12:30:00Z
➕

Cấu hình Webhook

ℹ️ Hướng dẫn:

Dữ liệu GET nhận về bao gồm các tham số:

message:
Nội dung tin nhắn
type:
Loại thông báo (sms, notification)
source:
Nguồn (Vietcombank, Techcombank..)
